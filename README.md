pajarito-fn v1.0.0
==================

The pajarito-fn library exported as [Node.js](https://nodejs.org/) modules.

Installation
------------
Using npm:

```
$ npm i -g npm
$ npm i pajarito-fn
```

In Node.js:

```Javascript
// load full module.
const pFn = require('pajarito-fn');
```

See [the package source](https://github.com/PajaritoMoyqi/NPM-pajarito-fn) for more details.
